## 📌 关于 Agent Workflow

Agent Workflow 是指将多个 Agent 或 AI 组件按照特定的流程编排起来，形成自动化工作流的系统。常见的 Workflow 工具包括：

- **Dify** - 可视化 LLM 应用开发平台
- **n8n** - 工作流自动化工具
- **Flowise** - 拖拽式 LLM 应用构建器
- **LangFlow** - LangChain 可视化编排工具
